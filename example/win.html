Child window

The parent window of this child window will fire the following events:

childwindowrefreshed

When the user refreshes this child window. The instance object of this child window will be passed in the property 'childWIndow' of the event object.

childwindowclosed

When the user closes this child window. The instance object of this child window will be passed in the property 'childWIndow' of the event object.

Check the console of the parent window to see it in action
